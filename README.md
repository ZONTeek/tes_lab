# Info

TES lab test application

# Extra step

At project root level add .env with the following line
`API_URL=https://swapi.dev/api/`

# Start

NPM:

```sh
npm i
npm start
npm android
```

Yarn:

```sh
yarn
yarn start
yarn android
```

# Requirements

- Node.js ^14.8
- Android SDK 11 (API level ^30)
- Java SDK ^11.0
